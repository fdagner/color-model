RGB-Farbmodell

Das RGB-Modell steht für "Rot, Grün, Blau" und ist ein additives Farbmodell, das in der Darstellung von Farben verwendet wird. In diesem Modell werden Farben durch die Kombination von verschiedenen Intensitäten der Grundfarben Rot, Grün und Blau erzeugt.

Die Intensität jeder Grundfarbe wird üblicherweise auf einer Skala von 0 bis 255 gemessen, wobei 0 für keine Intensität steht und 255 für maximale Intensität.Zum Beispiel ergibt die Kombination von voller Intensität (255) von Rot, Grün und Blau die Farbe Weiß, während keine Intensität (0) für alle drei Farben die Farbe Schwarz ergibt.

Farben kombinieren:

Gib die RGB-Werte ein, um die Farbe zu generieren. Der Hexadezimalcode* und die dargestellte Farbe werden automatisch aktualisiert.

* Hexadezimalcode ist ein Zahlensystem mit der Basis 16, das Zahlen mit den Zeichen 0-9 und den Buchstaben A-F repräsentiert. Im Zusammenhang mit HTML wird Hexadezimalcode häufig für die Darstellung von Farben verwendet, insbesondere im RGB-Farbmodell. In HTML können Farben durch Hexadezimalcode dargestellt werden, indem der Code mit einem Hashtag (#) beginnt, gefolgt von sechs Hexadezimalziffern. Jede Ziffer repräsentiert dabei einen Teil der Farbe im RGB-Modell (zwei Ziffern für Rot, zwei für Grün und zwei für Blau).
